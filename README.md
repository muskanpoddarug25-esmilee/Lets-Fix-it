   # LetsFixIt — Hyperlocal Civic & Infrastructure AI Portal

LetsFixIt is a hyperlocal, AI-powered civic platform that bridges the gap between citizens and municipal authorities, enabling residents to report, verify, and track local infrastructure issues like potholes, water leakages, broken streetlights, and waste management concerns.

Built for the **Vibe2Ship SGIP Hackathon 2026**.

## Problem Statement
Community Hero — Hyperlocal Problem Solver

## Key Features
- AI-powered issue reporting and categorization (Gemini)
- AI Damage Cost & Risk Estimator
- Smart Duplicate Detection
- Commute Safety Guard
- Real-time issue tracking with community upvoting
- Live AI-generated risk heatmap
- Gamification and leaderboards
- LetsFixIt AI Civic Assistant chatbot
- Secure citizen and government admin portals
- Government Admin Dashboard with triage and analytics

## Tech Stack
React (TypeScript), Vite, Tailwind CSS, Node.js, Express, Firebase Firestore, Firebase Authentication

## Google Technologies Used
Google AI Studio, Google Gemini AI, Google Maps Platform, Firebase Authentication, Firebase Cloud Firestore, Google Cloud Run
